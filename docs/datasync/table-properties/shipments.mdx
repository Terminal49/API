---
title: Shipments
---
The `shipments` table contains 1 row per shipment (`shipment_id` is the unique key).

A shipment contains 1 or more containers.


| COLUMN NAME                     | DESCRIPTION                                                                                     | DATA TYPE | PRIMARY KEY |
|---------------------------------|-------------------------------------------------------------------------------------------------|-----------|-------------|
| shipment_id                     | Shipment ID. This is the unique key of the table.                                               | text      | Yes         |
| shipping_line_scac              | Standard carrier alpha numeric code of the shipping line                                        | text      | No          |
| shipping_line_name              | Name of the shipping line                                                                       | text      | No          |
| bill_of_lading_number           | Shipment number from the tracking request                                                       | text      | No          |
| normalized_number               | The normalized version of the shipment number used for querying the carrier                     | text      | No          |
| reference_numbers               | Reference numbers of the shipment, contatenated                                                 | text      | No          |
| tags                            | Tags added to the shipment, sorted by alphabetical order, concatenated and separated by a comma | text      | No          |
| customer_id                     | Account ID of the customer                                                                      | text      | No          |
| customer_name                   | Name of the customer                                                                            | text      | No          |
| pol_locode                      | Port of Lading UN/LOCODE                                                                        | text      | No          |
| pod_locode                      | Port of Discharge UN/LOCODE                                                                     | text      | No          |
| pod_terminal_firms_code         | Port of Discharge terminal firms code                                                           | text      | No          |
| destination_locode              | Destination UN/LOCODE                                                                           | text      | No          |
| destination_terminal_firms_code | Destination terminal firms code                                                                 | text      | No          |
| pol_atd_at                      | Port of Lading Actual Time of Departure, as a UTC timestamp                                     | timestamp | No          |
| pol_etd_at                      | Port of Lading Estimated Time of Departure, as a UTC timestamp                                  | timestamp | No          |
| pod_eta_at                      | Port of Discharge Estimated Time of Arrival, as a UTC timestamp                                 | timestamp | No          |
| pod_arrived_at                  | Port of Discharge Actual Time of Arrival, as a UTC timestamp                                    | timestamp | No          |
| pod_voyage_number               | Voyage number of the vessel that arrived or will arrive at the POD                              | text      | No          |
| pod_vessel_name                 | Name of the vessel that arrived or will arrive at the POD                                       | text      | No          |
| pod_vessel_imo                  | IMO of the vessel that arrived or will arrive at the POD                                        | text      | No          |
| line_tracking_last_succeeded_at | When the shipment information was last refreshed from the shipping line, as a UTC timestamp     | timestamp | No          |
| line_tracking_stopped_at        | When the tracking of the shipment stopped, as a UTC timestamp                                   | timestamp | No          |
| line_tracking_stopped_reason    | Reason why the tracking of the shipment stopped                                                 | text      | No          |
| created_at                      | When the shipment was added, as a UTC timestamp                                                 | timestamp | No          |
| updated_at                      | When the shipment was last updated, as a UTC timestamp                                          | timestamp | No          |
