# Interface: TransportConfig

## Properties

| Property | Type | Description |
| ------ | ------ | ------ |
| <a id="property-accountid"></a> `accountId?` | `string` | - |
| <a id="property-apitoken"></a> `apiToken` | `string` | - |
| <a id="property-baseurl"></a> `baseUrl` | `string` | - |
| <a id="property-fetchimpl"></a> `fetchImpl?` | (`input`, `init?`) => `Promise`\<`Response`\> | - |
| <a id="property-maxretries"></a> `maxRetries?` | `number` | - |
| <a id="property-timeoutms"></a> `timeoutMs?` | `number` | Per-request timeout in milliseconds. Defaults to [DEFAULT\_REQUEST\_TIMEOUT\_MS](/sdk/reference/client/transport/variables/DEFAULT_REQUEST_TIMEOUT_MS). Set to `0` to disable the timeout. |
