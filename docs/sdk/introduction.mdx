## What you can do

- Track containers, bookings, or bills of lading
- List shipments and containers with filters
- Pull transport events and routing details
- Receive status updates via webhooks

## Before you start

You need a Terminal49 API key. Create one in the developer portal and store it in `T49_API_TOKEN`.
