## Query filters

| Filter | Description |
|--------|-------------|
| `filter[entity_type]` | Filter by entity type (`Shipment` or `Cargo`) |
| `filter[data_type]` | Filter by data type |
| `filter[display_name]` | Filter by display name (prefix match) |
